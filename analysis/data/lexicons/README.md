# Lexiques

## NRC-VAD (Valence–Arousal–Dominance)

Source : Mohammad, NRC Canada. http://saifmohammad.com/WebPages/nrc-vad.html

Le fichier `nrc-vad/NRC-VAD-Lexicon/OneFilePerLanguage/French-NRC-VAD-Lexicon.txt` contient les scores VAD pour ~20 000 termes en français (traduction automatique depuis l’anglais, août 2022). Format : tabulations, colonnes English Word, Valence, Arousal, Dominance, French Word.

Usage non commercial, recherche et éducation. Citer Mohammad (ACL 2018) si utilisation.
