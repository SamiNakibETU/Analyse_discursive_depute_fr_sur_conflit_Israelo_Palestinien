# Évaluation projet - Postes Intelligence économique / Analyste politique

**Rôle évalué** : Analyste en intelligence économique, analyste politique, consultant en risk & geopolitical analysis.

---

## Note globale : **16/20** - Très bon alignement

Le projet est **au-dessus du lot** pour ces profils. La combinaison données politiques + méthodologie quantitative + cadrage théorique correspond exactement aux attentes.

---

## Forces majeures (alignées IE / Analyste politique)

| Critère | Avis |
|---------|------|
| **Sources primaires** | Données réelles (députés, tweets, interventions AN) vs. agrégats médiatiques |
| **Cadre événementiel** | Fenêtres autour de CIJ, Rafah, CPI : lecture **temporalité/événements** typique IE |
| **Mapping des acteurs** | Blocs, cibles des critiques (target_primary), identification des « movers » |
| **Indicateurs de convergence** | Diffusion du lexique cessez-le-feu, évolution par batch : **anticipation de positions** |
| **Cas négatif** | Droite comme contre-exemple : démarche comparative structurée |
| **Cadres théoriques** | Entman (framing), Kostikova et al., Monroe fightin' words : ancrage littérature |

---

## Ce qui renforce le profil IE

- **Veille structurée** : pipeline reproductible (Makefile, config centralisée)
- **Multi-sources** : Twitter + AN pour triangulation
- **Diffusion temporelle** : décalage de 14 mois entre blocs sur le lexique CF
- **Risques identifiables** : décalage positions implicites vs. explicites, polarisation
